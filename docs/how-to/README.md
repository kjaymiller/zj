# How-to guides

Task-oriented guides for things you already understand and want to get done. For
a guided introduction start with the [tutorial](../tutorial.md); for exact values
see the [reference](../reference.md).

- [Set up remote web sessions](remote-sessions.md) — add remote zellij servers to the picker, with token and TLS handling.
- [Add GitHub orgs to the picker](github-orgs.md) — list an owner's repos as `gh:owner/repo` and clone-on-demand.
- [Troubleshooting](troubleshooting.md) — fixes for the common failure modes.
